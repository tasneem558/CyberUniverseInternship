# Day 20 – Learning Summary & Next Plan

## 1. My Top 5 Learnings

1. **Linux basics** are very important, especially users, permissions, and using the command line.
2. I learned how to identify **attack surfaces** like cracks, unsafe software, and unnecessary services.
3. I understood the idea of **Defense in Depth** and why one layer of security is not enough.
4. I learned that **phishing and social engineering** target people, not only systems.
5. I learned that small daily habits matter, like using a **standard user account instead of admin** all the time.


## 2. 3 Mistakes I Will Not Repeat

1. Using **Admin/Root account** for daily use.
2. Disabling **antivirus** just to run unsafe or cracked software.
3. Focusing only on using tools without understanding the **concept behind them**.


## 3. My Next 30-Day Learning Plan

- **Week 1–2:** Focus on **Networking Basics** (IP addressing, protocols, ports, and traffic flow).
- **Week 3:** Learn **Active Directory basics** because it is important in company environments.
- **Week 4:** Start reviewing **Security+ fundamentals** and practice simple labs related to blue team topics.


## 4. Focus Area

My main focus is becoming a **SOC Analyst**.

I want to build a strong path starting with **networking**, then **Active Directory**, and then continue improving my security fundamentals step by step.
