# Day 17 – Self-Directed Learning

## 1. What I chose

I chose the topic of **Phishing Attacks**.


## 2. Why I chose it

I chose this topic because phishing is very common and it is related to things we learned before like malware and unsafe downloads.

I also wanted to understand how attackers trick users, not only how they attack systems.


## 3. What I learned

I learned that phishing is when attackers try to trick people using fake emails, messages, or links.

They often try to make the user panic or click quickly.

I also learned that I should always check the sender, be careful with links, and avoid downloading files from unknown sources.


## 4. How it connects to my future goals

This topic is useful for my future goals because I want to improve in cybersecurity and understand different types of attacks.

It also helped me understand that security is not only about tools, but also about user awareness.
