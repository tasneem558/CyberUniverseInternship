# Day 21 – Final Reflection

## 1. One major challenge I faced during the program

One major challenge I faced was changing the way I think.

At first, I was thinking more like a normal user, not like someone working in cybersecurity.

It was also a challenge for me to understand Linux better and get used to the command line.

I also realized that some habits I thought were normal, like using cracks or using admin account all the time, were actually security risks.


## 2. How I overcame or handled it

I handled this by practicing more and trying to understand things step by step.

I didn’t just read the tasks, I tried to apply what I learned.

I also used simple examples and analogies to help myself understand concepts better.

I started checking my own device, removing risky software, and changing some bad habits.


## 3. One change in how I learn now

Now I learn in a more structured way.

Before, I used to focus more on quick answers or just how to use a tool.

Now, I focus more on understanding the basics and knowing why something works, not only how to use it.


## 4. Why I believe I deserve to complete this program

I believe I deserve to complete this program because I stayed consistent and took the tasks seriously.

I didn’t only finish the tasks, but I also changed the way I think and improved my learning approach.

I became more aware of security risks, more focused on fundamentals, and more serious about my goal of becoming a SOC Analyst.
