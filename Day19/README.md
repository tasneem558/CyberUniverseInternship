# Day 19 – Perspective Shift

## 1. One belief I had before starting

Before starting this program, I thought cybersecurity was mostly about hacking tools and using programs to do everything.


## 2. Why that belief is not fully correct

Now I understand that tools are not enough by themselves.

If I don’t understand the basics, like how permissions work in Linux or how a firewall works, then using tools alone will not make me good in cybersecurity.

Cybersecurity needs understanding, practice, and the right mindset, not only tools.


## 3. One reason instant results are not realistic

Instant results are not realistic because cybersecurity takes time.

There are many things to learn, and you need practice, not just watching videos.

Also, the field keeps changing, so learning is not something you finish quickly.


## 4. One realistic expectation I now have

Now I expect that learning cybersecurity will be a continuous process.

I know I need to keep learning step by step, practice regularly, and improve over time instead of expecting fast results.
