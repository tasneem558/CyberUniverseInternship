# Day 16 – Defensive Thinking

## 1. One attack surface from Day 15

One attack surface I chose from Day 15 is **cracked software or programs from unofficial websites**.


## 2. At least two defenses for that attack surface

### 1. Downloading software only from trusted or official sources
This is safer because official sources are less risky than random websites.  
It reduces the chance of downloading harmful or modified files.

### 2. Keeping antivirus turned on and updated
Antivirus can help detect harmful files before they run or warn me if something looks suspicious.  
Keeping it updated also helps protect against newer threats.


## 3. Why one defense alone is not enough

One defense alone is not enough because if one layer fails, the system can still be at risk.

For example, even if I use antivirus, some harmful files might still get through.  
And even if I download from a trusted source, it is still better to have extra protection.

That’s why using more than one defense is important.


## 4. One defense I previously underestimated

One defense I previously underestimated was **using a normal user account instead of admin all the time**.

Before, I thought using admin access all the time was easier and normal.  
Now I understand that using a normal user account can help reduce damage if a harmful file tries to change something in the system.
