# Day 14 – Weekly Review & Reflection

## 1. One concept that is now clearer than before

One concept that is much clearer for me now is how **IP addresses** work and how much they can reveal.

Before these tasks, I knew the IP address was important, but I didn’t really think about how it can show location and ISP information.  
After doing the **VPN** task and the **Wireshark** task, I understood better how the real IP can be hidden by a VPN and how network traffic can still show a lot of useful information.

## 2. One topic I still find difficult

The topic I still find a bit difficult is **Wireshark packet analysis**.

I can now recognize some protocols like **DNS** and **TCP**, and I can use filters, but the number of packets is still a little overwhelming.  
Sometimes it’s hard to know which packets are the most important and how to fully understand all the details inside them.

## 3. One improvement in how I approach learning cybersecurity

One thing that improved is that I became more **patient while troubleshooting**.

Before, if something didn’t work quickly, I used to feel stuck.  
Now I try to check step by step, like reviewing the command, checking the firewall status, or making sure I’m using the correct interface.

This made me realize that cybersecurity is not just about using tools, but also about thinking step by step and troubleshooting when something doesn’t work.

## 4. One connection I noticed between two different tasks

One strong connection I noticed was between **Nmap** and **Wireshark**.

With **Nmap**, I was checking open ports and trying to see what services are available.  
With **Wireshark**, I was able to see the actual traffic moving through the network.

So for me, **Nmap** helped me understand what is open, and **Wireshark** helped me understand what is happening on the network.
